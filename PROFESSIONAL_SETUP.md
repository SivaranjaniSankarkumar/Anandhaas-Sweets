## üöÄ Quick Start

### Prerequisites
- Node.js 16+ 
- npm or yarn
- Modern web browser

### Installation
```bash
# Navigate to project directory
cd sweets-dashboard

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

### Backend Setup
```bash
# Navigate to backend directory
cd backend

# Install Python dependencies
pip install -r requirements.txt

# Start Flask server
python app.py
```

## üîß Customization

### Brand Colors
Update `tailwind.config.js` primary colors to match your brand:
```javascript
primary: {
  500: '#YOUR_BRAND_COLOR',
  600: '#DARKER_SHADE',
}
```

### Logo Replacement
Replace logo files in:
- `src/assets/images/logo.png`
- `public/logo1main.png`
- `public/favicon.ico`

- Update company name in components
- Modify dashboard metrics and data
- Customize navigation items
